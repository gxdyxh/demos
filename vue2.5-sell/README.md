# Use

开发

    下载代码  git clone
	安装依赖  yarn install / npm install
	运行项目  yarn serve / npm run serve
    浏览器预览 http://localhost:8080

打包

    打包代码 yarn build / npm run build
	node代理 node prod.server.js
	浏览器预览 http://localhost:8900
